# Fake Store Products - Interactive Data Explorer

## Overview
A responsive single-page web application built with React + Vite that fetches and displays products from the Fake Store API. Users can search for products and browse categories seamlessly.

## Features
- Display products in a responsive grid layout
- Functional search bar to filter products
- Loading and error states for API calls
- Component-based architecture using React
- Fully responsive design (mobile, tablet, desktop)

## Technologies Used
- React.js
- Vite
- JavaScript (ES6+)
- Tailwind CSS
- Git & GitHub

## API Used
https://fakestoreapi.com/

## Setup Instructions

1. Clone the repository:
git clone https://github.com/Krishna0812478/data-explorer.git


2. Install dependencies:

npm install


3. Run the development server:

npm run dev


4. Open browser and go to:

http://localhost:5173/
